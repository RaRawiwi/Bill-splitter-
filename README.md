# Bill-splitter-
A small web app for solving the eternal “ใครยังไม่โอน” problem after eating with friends 🍜

## Features
- Add food items and their prices
- Add participant names
- Enter which menu items each person ate
- Automatically calculate each person's total payment
- Simple and clean user interface

## Tech Stack
- Python
- Flask
- HTML
- CSS
